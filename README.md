🎓 Freelancer Marketplace for Students

📖 Description

This project is a web-based application that allows students to collaborate through a freelancer marketplace. Students can post projects, apply as freelancers, communicate with each other, and provide feedback.

---

💡 Features

- 🔐 Login system
- 📋 View projects
- 📤 Post new projects
- 📝 Apply for projects
- ✅ Accept / Reject applications
- 💬 Chat between students
- ⭐ Ratings and reviews
- 📊 Dashboard with graphs

---

🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Excel (used as database)

---

📁 Project Structure

student_marketplace/
│
├── app.py
├── data.xlsx
├── requirements.txt
├── README.md

---

▶️ How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the application:
   python -m streamlit run app.py

3. Open in browser:
   http://localhost:8501

---

🎯 Objective

To create a simple freelancer platform where students can collaborate, work on projects, and gain practical experience.

---

🧠 Conclusion

This project demonstrates how Excel can be used as a backend database along with Streamlit to build an interactive and user-friendly web application.
